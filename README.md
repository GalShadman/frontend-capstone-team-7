# frontend-capstone-team-7

DJ Mixer

System Description
DJ Mixer is a web project that simulates a music mixing platform. The website includes a landing page, a login page, a home page with mixes and playlists, a mix creation page with a sound library and instruments, an About page presenting the vision of the platform, and an 'Under Construction' page that appears for features that are not yet fully implemented.

Technologies Used
HTML – Structure of the website pages.
CSS – Custom styling for each page (separate CSS file per page).
JavaScript – Page interactions (Login.js / CreateMixPage.js / About.js).
Google Fonts – Use of fonts such as Nunito and Orbitron.
Bootstrap (Navbar, Offcanvas) – Buttons, grid system, responsiveness, and layout components.

Features Summary
DJ MIXER (Homepage)
Landing page with title and slogan.
"Start Mixing" button leading to the Login page.
Footer with a Support link.

LOGIN Page
Login form with Email and Password fields.
Show/Hide password option (icon toggle).
"Remember me" checkbox and "Forgot password" link.
"Continue with Apple / Google / Facebook" buttons leading to the Construction page.
"Join us!" link for future premium registration.

MAIN PAGE
Top Navbar with side menu (Offcanvas).
Navigation links: Home / Create a Mix / About.
Search bar.
Mix display by categories: Your Mixes, Mixes You Like, Mixes For You (image gallery).
"Create a Mix" button leading to the mix creation page.

CREATE MIX PAGE
Sounds Library with instrument buttons.
Each instrument includes a data-sound attribute for playing an audio file.
Mixer Controls area with sliders (mixer-style design).
Player controls: Back / Play / Forward buttons.
Dynamic Tracks area (main-track) for adding tracks using JavaScript.
"Save Mix" button leading to the Construction page.

ABOUT Page
Content page presenting the purpose of the platform, who we are, and why the system exists.
Modern design with cards, icons, and SVG elements.
Action buttons: "Join the vibe" and "Explore features".

Construction Page (Under Construction)
Temporary page shown when users click on features that are not yet active.
"Under Construction" message inviting users to return later.
"Create a Mix" button that redirects back to the mix creation page.

How to Run the Project
Download the project folder.
Open index.html in your browser.
Navigate through the website using the buttons and navigation menu:
Start Mixing → Login → Home → CreateMixPage / About.
